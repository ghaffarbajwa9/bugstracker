# Bugs Tracker System
* Requirements and specification for this system:
  * Rails 7.1.2
  * Ruby 3.2.1
  
* System dependencies
  * activeadmin (3.1.0)
  * cancancan (3.5.0)
  * carrierwave (3.0.5)
  * devise (4.9.3)
  * sass-rails (6.0.0)
  * pg (1.5.4)
    
* Installations and configurations
  * ```console
    > git clone https://github.com/ghaffarbajwa9/bugstracker.git
    > cd bugstracker
    > bundle (if required)
    > bin/rails db:create
    > bin/rails db:migrate
    > bin/rails db:seed
    > bin/rails server
